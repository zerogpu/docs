ZeroGPU keeps the catalog small and purpose-built. Choose a model based on the task and the output type you need.

## Catalog

| Model ID | Task | Output type | Best for |
| --- | --- | --- | --- |
| `zlm-v1-iab-classify-cloud` | IAB ad signals extraction | JSON string | Ad targeting, classification, content labeling |
| `zlm-v1-summary-cloud` | Summarization | Plain text | UI previews, dashboards, internal tools |
| `TBD` | ZLM v1 (Coming soon) | TBD | Placeholder for upcoming model |

## How to choose

- Use `zlm-v1-iab-classify-cloud` when you need structured categories, keywords, and intent scores.
- Use `zlm-v1-summary-cloud` when you want a concise summary of a text input.
- Watch the catalog for new models as the closed beta expands.
