OpenAI-compatible APIs for fast, specialized models built for classification and summarization. Call ZeroGPU with your own content and get structured ad signals or clean summaries back.

## What you can do

<Cards>
  <Card title="Extract IAB ad signals" icon="tag" href="/models/zlm-v1-iab-classify-cloud">
    Classify content into IAB categories, audience segments, topics, and intent signals.
  </Card>
  <Card title="Summarize text" icon="align-left" href="/models/zlm-v1-summary-cloud">
    Generate concise summaries for UI previews, dashboards, and content pipelines.
  </Card>
</Cards>

## How it works (at a high level)

1. Join the waitlist to get closed-beta access.
2. Create an API key and project ID in the dashboard.
3. Choose a model from the catalog.
4. Call the API and parse the response.
