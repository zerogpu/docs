---
title: "CrewAI"
description: "Configure CrewAI with a ZeroGPU base URL and headers."
---

CrewAI supports OpenAI-compatible providers when you can override the base URL and inject headers.

## Steps

1. Set the base URL to `https://api.zerogpu.ai`.
2. Add `x-api-key` and `x-project-id` headers.
3. Choose a ZeroGPU model name.

## Pseudocode

```text
agent = OpenAICompatible(
  base_url="https://api.zerogpu.ai",
  headers={"x-api-key": "...", "x-project-id": "..."},
  model="zlm-v1-summary-cloud"
)
```
