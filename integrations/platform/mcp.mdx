---
title: "MCP"
description: "Call ZeroGPU from MCP tools using OpenAI-compatible endpoints."
---

MCP tools can call OpenAI-compatible endpoints. ZeroGPU does not provide an official MCP server yet, but you can configure your own client or proxy.

## Conceptual server config

```json
{
  "provider": "openai-compatible",
  "base_url": "https://api.zerogpu.ai",
  "headers": {
    "x-api-key": "<YOUR_API_KEY>",
    "x-project-id": "<YOUR_PROJECT_ID>"
  },
  "model": "zlm-v1-summary-cloud"
}
```
