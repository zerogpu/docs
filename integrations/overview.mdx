ZeroGPU is OpenAI-compatible, so most tools that support base URL overrides can call it today. Official, first-class SDKs and integrations are planned.

## What works now

- Point the base URL to `https://api.zerogpu.ai`
- Add required headers: `x-api-key` and `x-project-id`
- Use a model name from the [Model Catalog](/models/overview)

See SDK and platform notes for wiring details.
