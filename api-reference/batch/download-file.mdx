Download the raw JSONL body for an input file you uploaded or for `output_file_id` / `error_file_id` from a [completed batch](/api-reference/batch/retrieve-batch).

Response is streamed as `application/jsonl`. Full reference: [Files API](/docs/batch/files-api).
