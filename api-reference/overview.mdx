All ZeroGPU APIs are OpenAI-compatible and require authentication headers.

## Base URL

`https://api.zerogpu.ai`

## Required headers

- `x-api-key`: your API key
- `x-project-id`: your project ID

## Endpoints

- [`POST /v1/responses`](/api-reference/responses)
- [`POST /v1/chat/completions`](/api-reference/chat-completions)
